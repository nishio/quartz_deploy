ã€€In addition, the operator new is provided to make it easier to write processes using prototypes. When the function f is called with new, the following four processes are performed.

- (a) Create a new object x
- (b) Change the prototype of the created object x to the prototype of the function f
- (c) Put the created object x into this and execute the body of function f.
- (d) Return object x

ã€€We will look at this in actual code.

ã€€This code first adds a new function named push to the prototype of function Counter in (2). Next, in (3), new Counter() performs the processes described in (a) through (d) above. First, an empty new object is created ((a)). Then, the prototype of that object is changed to the prototype of Counter ((b)). This prototype contains push. Next, Counter is called with this object in this ((c)), and since it is written as (1) in Counter, a value of 0 is added to this object here, with the name count. Finally, this "object with count in it that has a prototype with push in it" is returned ((d)).
