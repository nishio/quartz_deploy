First, let's briefly talk about what an EM algorithm is. A situation where, in addition to the observable random variable X, there is also an unobservable random variable Z. In order to represent this situation, you assume some probability model for yourself. You want to choose the best parameter Î¸ of that probability model based on the observed data X. This is the objective. In other words, you want to find Î¸ that maximizes p(X|Î¸).

Unfortunately, however, the expression for p(X | Î¸) cannot be easily maximized (if it could, there would be no need to use the EM algorithm). And fortunately, maximizing the log-likelihood ln p(X, Z | Î¸) of the complete data is easy.

- p(X | Î¸) cannot be easily maximized, but by Jensen's inequality we can say that it is greater than some expression L(q(Z), Î¸)
- Stopping Î¸ and maximizing L for q(Z) can be done by setting q to p (E-step), since the remainder is KL(q|||p) if we cull ln p(X|Î¸), which is unrelated to Z, out of L.
- Stopping q(Z) and maximizing L with respect to Î¸ means that if we cobble together a denominator from L that is irrelevant to Î¸...er...hmmm...what does the Q function mean?
