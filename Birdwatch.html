Matrix Factorization (MF) is a technique for extracting potential user and annotation characteristics from user and annotation evaluation data. In this paper, MF is used to evaluate the quality of Birdwatch‚Äôs annotations as follows

  • Create a user x annotation rating matrix. Each element represents whether or not a user rated a particular annotation as ‚Äúuseful‚Äù.
